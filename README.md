# Chess Puzzles Menu Item

A macOS menu bar application for solving chess puzzles. The app provides daily chess puzzles from the Lichess puzzle database, allowing you to practice and improve your chess skills directly from your menu bar.

## Features

- Daily chess puzzles from Lichess database
- Multiple difficulty levels
- Statistics tracking
- Timer for puzzle solving
- Beautiful chess board with customizable colors
